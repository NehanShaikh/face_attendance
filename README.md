## ğŸ§© System Workflow (Methodology)

1. **Admin** creates user credentials for faculty and students.  
2. **Students** log in using provided credentials and self-register.  
3. **Faculty** log in to view assigned students and subjects.  
4. During class, faculty click the **"Run Face Recognition"** button on their dashboard.  
5. The system detects faces via a camera/CCTV feed using **MTCNN**.  
6. Extracted embeddings are compared with stored embeddings using **FaceNet**.  
7. Matched faces are marked as **Present** in the **attendance** table.  
8. Attendance details are instantly reflected in the **student dashboard**.
